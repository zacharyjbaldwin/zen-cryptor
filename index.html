Introduction

Everything that you need to know about Zen-Cryptor.

About

Zen-Cryptor is a lightweight tool created with NodeJS and Electron used to encrypt text and files, and decrypt text and files.

License

Zen-Cryptor is open source and licensed under the MIT license.

File Encryption

Encrypting files obfuscates the contents of the files chosen based on your password and the encryption algorithm you have chosen.

You are able to batch-encrypt files, meaning that you can encrypt multiple files at once.

Files that are encrypted with Zen-Cryptor are converted to '.zc' files. For example, if one were to encrypt 'sunny-beach-home.jpg' it would be converted to 'sunny-beach-home.jpg.zc'.

Encrypt File(s)

File Decryption

Decrypting files deobfuscates the contents of the files encrypted based on the password and encryption algorithm that it was encrypted with.

You are able to batch-decrypt files, meaning that you can decrypt multiple files at once.

At the moment, you can only decrypt '.zc' files, which have been encrypted with Zen-Cryptor.

Encrypted '.zc' files are converted and returned to their original filetype. For example, if one were to decrypt 'sunny-beach-home.jpg.zc' it would be converted to 'sunny-beach-home.jpg'.
